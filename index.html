Guia por si te perdiste con tus Repos

Pasos, primero lo primero

1. Crear una Pagina HTML

Abre Visual Studio Code y crea una nueva carpeta para tu proyecto. Luego, dentro de esa carpeta, crea un nuevo archivo llamado index.html.

No olvides guardar el archivo despues de escribir tu codigo

2. Inicializar un Repositorio Git

Abre una terminal integrada en Visual Studio Code (puedes hacerlo con el atajo Ctrl + `) y navega hasta la carpeta de tu proyecto. Una vez allí, inicializa un nuevo repositorio git con el siguiente comando: git init

3. Añadir Archivos y Hacer el Primer Commit

Añade todos los archivos del proyecto al repositorio y haz tu primer commit: 'git add .' luego 'git commit -m "Primer commit - Página HTML creada"'

4. Crear un Repositorio en GitHub

Ve a GitHub y crea un nuevo repositorio. Dale un nombre a tu repositorio (por ejemplo, mi-pagina-html) y asegúrate de no inicializarlo con un README, .gitignore o una licencia.

5. Conectar el Repositorio Local con GitHub

Regresa a la terminal en Visual Studio Code y añade el repositorio remoto con el siguiente comando, reemplazando y con tu nombre de usuario y el nombre del repositorio que acabas de crear en GitHub:

git remote add origin

6. Subir los Cambios a GitHub

Finalmente, sube tus cambios al repositorio remoto con los siguientes comandos:

git branch -M main

git push -u origin main

7. Verifica en GitHub

Ve a tu repositorio en GitHub y verifica que los archivos se han subido correctamente. Deberías ver tu archivo index.html y cualquier otro archivo que hayas incluido en el repositorio.

¡Y eso es todo! Has creado una página HTML y la has subido a GitHub usando Visual Studio Code
